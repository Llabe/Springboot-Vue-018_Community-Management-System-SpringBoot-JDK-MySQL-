# SpringBoot-Vue-130社团管理系统

#### Description
基于 SpringBoot + MyBatis-Plus + Shiro 的 B/S 架构社团管理系统，包含管理员、社长和用户三级权限体系，实现社团创建审核、活动报名、经费审批等核心功能，采用 MySQL 数据库保障数据安全。
